# ADR-0001: Core Modeとして異端者モードを採用する

## Status

Accepted

## Date

2026-07-12

## Context

Project Personaでは、MBTIを演じることで相手の考え方や価値観の違いを楽しむ正体隠匿ゲームを目指している。

初期案では、人狼ゲームをベースとして、

- 誰が人狼なのか
- 誰が指定されたMBTIを演じているのか

という二重の推理要素を持つゲームを検討していた。

しかし、初期プレイヤーにとっては複数の推理要素を同時に理解する必要があり、Project Personaの本質である「人格を演じる楽しさ」が伝わりにくくなる可能性がある。

## Decision

Project Personaの基本ルール（Core Mode）として、異端者モードを採用する。

異端者モードでは、

1. 全プレイヤーに本来のMBTIを割り当てる
2. 全プレイヤーにMBTIを公開する
3. 1人だけ異なるMBTIを演じる「異端者」を設定する
4. 会話を通じて異端者を推理する

というゲーム体験を中心とする。

人狼モードは、異端者モードを発展させた上級者向け拡張ルールとして扱う。

## Reasoning

異端者モードを採用する理由：

- ルール説明がシンプルで初心者が参加しやすい
- MBTIを演じる楽しさが直接伝わる
- Project Personaの独自性である「人格推理」を中心にできる
- 人狼経験がないプレイヤーでも楽しめる
- プレイ後の振り返りや会話が生まれやすい

## Core Experience

プレイヤーが感じるべき体験：

- 「この人、本当にこのMBTIなのか？」
- 「この発言はそのタイプらしい？」
- 「まさかこの人が異端者だったとは！」

という驚きと発見。

## Consequences

### Positive

- 初見プレイヤーでも理解しやすい
- MVP開発範囲を小さくできる
- MBTIというテーマを最大限活用できる

### Negative

- 人狼ゲームとして期待するプレイヤーには物足りない可能性がある
- 嘘をつく要素は弱くなる

## Future Considerations

基本ゲーム体験が成立した後、以下の拡張を検討する。

- 人狼モード
- 複数異端者モード
- 特殊役職
- AIプレイヤー
- オンライン対戦
