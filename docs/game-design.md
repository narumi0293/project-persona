# Game Design Document

## 1. Overview

## Game Title

Project Persona（仮）

## Genre

MBTI人格ロールプレイ型 正体推理ゲーム

## Concept

Project Personaは、MBTIを診断するゲームではなく、演じることで理解し、対話を楽しむ新しいコミュニケーションゲームである。

プレイヤーは自身に割り当てられたMBTIタイプを理解し、その人格を演じながら他プレイヤーと会話する。

その中で、1人だけ異なるMBTIを演じる「異端者」を探し出す。

---

# 2. Vision

MBTIを診断するのではなく、演じることで理解し、対話を楽しむ新しい正体隠匿ゲームをつくる。

Project Personaでは、勝敗だけではなく、

- 相手の考え方を想像する楽しさ
- 自分がどう見られているかを知る楽しさ
- 意外な発見を共有する楽しさ

を重視する。

---

# 3. Core Experience

プレイヤーに提供する体験：

## 演じる楽しさ

普段とは異なる人格を意識して会話する。

## 推理する楽しさ

発言や行動から、

「この人は本当にこのMBTIなのか？」

を考える。

## 発見する楽しさ

ゲーム終了後、

「そのタイプだったの！？」

という驚きや会話が生まれる。

---

# 4. Game Modes

## Core Mode: 異端者モード

Project Personaの基本ルール。

プレイヤー全員のMBTIを公開した状態で、1人だけ異なるMBTIを演じる異端者を探す。

---

## Future Mode: 人狼モード

異端者モードを発展させた上級ルール。

追加要素：

- 人狼役職
- 嘘をつく必要性
- 生存戦略
- 特殊能力

などを検討する。

---

# 5. Core Mode: 異端者モード

## Player Count

3〜6人

推奨人数：

5人

---

## Preparation

1. プレイヤーへ本来のMBTIを割り当てる
2. 全員のMBTIを公開する
3. 1人を異端者として選択する
4. 異端者へ演じる別のMBTIを指定する

---

## Example

開始時：

| Player | 本来のMBTI |
| --- | --- |
| A | INTJ |
| B | ENFP |
| C | ISFJ |
| D | ENTP |
| E | INFJ |

異端者：

C

本来：

ISFJ

演じる：

ENTJ

---

# 6. Game Flow

## Phase 1: Start

- プレイヤー参加
- MBTI公開
- 異端者決定

---

## Phase 2: Discussion

テーマに沿って全員で会話する。

会話例：

- 「休日の過ごし方」
- 「仕事で大切なこと」
- 「困った時の行動」
- 「理想のリーダー像」

など。

---

## Phase 3: MBTI Prediction

各ターン終了時、プレイヤーは他プレイヤーのMBTIを予想する。

表示形式：

例：

```
Aさんの予想

INTJ：3票
INFJ：1票
ENTP：1票
```

仕様：

- 得票数を表示する
- パーセント表示は行わない
- 投票結果は全員に公開する

---

## Phase 4: Continue

議論と予想を繰り返す。

---

## Phase 5: Result

最終的に異端者を公開する。

---

# 7. MBTI System

## Basic Design

MBTIは単なるタイプ名ではなく、ゲーム内人格情報として扱う。

各MBTIは以下の情報を持つ。

- タイプ名
- 基本特徴
- 行動傾向
- 会話傾向

---

## Example

### INTJ

特徴：

- 計画を立てることを好む
- 論理的に考える
- 長期的視点を持つ

会話傾向：

- 結論や理由を重視する
- 慎重に発言する

---

## Future Expansion

将来的には、MBTI特徴を利用した能力やゲーム要素を検討する。

例：

- 特定タイプの特徴を活かした能力
- MBTIカード
- 特殊ルール

ただしMVPでは追加しない。

---

# 8. Prediction System

Project Persona独自の主要システム。

## Purpose

プレイヤー同士の認識のズレをゲーム要素にする。

---

## Features

- 毎ターン予想する
- 結果を公開する
- 得票数を表示する

---

## Expected Experience

プレイヤーは、

「自分はこう見られているのか」

という新しい情報を得る。

その情報が次の会話や推理につながる。

---

# 9. Win Conditions

## Normal Player

異端者を見つけること。

---

## Heretic Player

最後まで異なるMBTIを演じ切ること。

または、他プレイヤーから自分の演じたMBTIとして認識されること。

---

# 10. Result Experience

ゲーム終了後に以下を表示する。

- 本来のMBTI
- 演じたMBTI
- 異端者だったか
- 他プレイヤーの予想結果

---

## Goal

勝敗だけではなく、

「なぜそう見えたのか」

を振り返れる体験を作る。

---
