# Project Persona

## Overview

Project Persona is a game development project
to create a new social deduction game.

The game combines:

- MBTI role play
- Werewolf game mechanics
- AI-powered interaction

## Vision

MBTIを診断するのではなく、
演じることで理解し、
対話を楽しむ新しい正体隠匿ゲームをつくる。

## Status

🚧 Planning Phase

## Development Philosophy

- Design before implementation
- Small prototype first
- Human + AI collaborative development
